> * Spring Boot Project (2.2.6.RELEASE).
> * Sample POJO.
> * Out of the box spring-boot-starter-actuator integration.
> * Out of the box spring-boot-starter-data-jpa integration.

> * Out of the box spring-boot-devtools integration.
